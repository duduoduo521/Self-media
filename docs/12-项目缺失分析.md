**已实现的完整链路**：

```
用户触发执行
  → 1. 验证任务归属 + 状态检查（Pending → Running）
  → 2. 获取用户 API Key（解密 MiniMax 密钥）
  → 3. 获取平台配置（Cookie、图片数等）
  → 4. 步骤一：AI 文本生成（调用 MiniMax 文本 API）
  → 5. 步骤二：AI 图片生成（根据文本生成配图）
  → 6. [视频模式] 步骤三：AI 视频生成 + 语音生成
  → 7. 步骤四：内容适配（根据各平台要求调整文案长度/标签/图片数）
  → 8. 步骤五：逐平台发布（调用 PublisherRegistry）
  → 9. 更新任务状态（Running → Completed / Failed）
  → 10. 推送进度事件（WebSocket / SSE）← 待实现
```
