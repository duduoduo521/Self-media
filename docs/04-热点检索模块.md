## 2. 统一热点模型

```rust
/// 热点来源枚举（独立于 Platform，包含知乎等非发布平台）
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, Eq, Hash)]
pub enum HotspotSource {
    Weibo,
    Douyin,
    Xiaohongshu,
    Bilibili,
    Toutiao,
    Zhihu,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Hotspot {
    pub title: String,
    pub hot_score: u64,
    pub source: HotspotSource,
    pub url: Option<String>,
    pub category: Option<String>,
    pub fetched_at: DateTime<Utc>,
}
```

> **修复说明**：使用独立的 `HotspotSource` 枚举替代 `Platform` 枚举，解决了知乎不在发布平台列表中的问题。

## 3. 热点服务

### 3.1 核心结构

```rust
pub struct HotspotService {
    http: Client,
    cache: Arc<Mutex<HotspotCache>>,
}

/// 热点缓存
struct HotspotCache {
    data: Vec<Hotspot>,
    last_fetch: Option<Instant>,
    ttl: Duration,  // 缓存有效期，默认 5 分钟
}

impl HotspotCache {
    fn new() -> Self {
        Self {
            data: Vec::new(),
            last_fetch: None,
            ttl: Duration::from_secs(300),
        }
    }

    fn is_valid(&self) -> bool {
        match self.last_fetch {
            Some(last) => last.elapsed() < self.ttl,
            None => false,
        }
    }
}
```

### 3.2 核心方法

```rust
impl HotspotService {
    pub fn new(http: Client) -> Self {
        Self {
            http,
            cache: Arc::new(Mutex::new(HotspotCache::new())),
        }
    }

    /// 获取所有平台热点（带缓存）
    pub async fn fetch_all(&self) -> Result<Vec<Hotspot>, AppError> {
        // 1. 检查缓存
        {
            let cache = self.cache.lock().unwrap();
            if cache.is_valid() {
                return Ok(cache.data.clone());
            }
        }

        // 2. 并发请求所有数据源
        let sources = vec![
            self.fetch_weibo(),
            self.fetch_douyin(),
            self.fetch_xiaohongshu(),
            self.fetch_bilibili(),
            self.fetch_toutiao(),
            self.fetch_zhihu(),
        ];
        let results = futures::future::join_all(sources).await;

        // 3. 合并结果（单源失败不影响整体）
        let mut all = Vec::new();
        for result in results {
            match result {
                Ok(hotspots) => all.extend(hotspots),
                Err(e) => {
                    tracing::warn!("热点数据源获取失败: {}", e);
                    // 单源失败不阻断
                }
            }
        }

        // 4. 排序去重
        all.sort_by(|a, b| b.hot_score.cmp(&a.hot_score));
        all.dedup_by(|a, b| a.title == b.title);

        // 5. 更新缓存
        {
            let mut cache = self.cache.lock().unwrap();
            cache.data = all.clone();
            cache.last_fetch = Some(Instant::now());
        }

        Ok(all)
    }

    /// 获取指定来源的热点
    pub async fn fetch_by_source(&self, source: HotspotSource) -> Result<Vec<Hotspot>, AppError> {
        match source {
            HotspotSource::Weibo => self.fetch_weibo().await,
            HotspotSource::Douyin => self.fetch_douyin().await,
            HotspotSource::Xiaohongshu => self.fetch_xiaohongshu().await,
            HotspotSource::Bilibili => self.fetch_bilibili().await,
            HotspotSource::Toutiao => self.fetch_toutiao().await,
            HotspotSource::Zhihu => self.fetch_zhihu().await,
        }
    }
}
```

impl HotspotService {
    async fn fetch_weibo(&self) -> Result<Vec<Hotspot>, AppError> {
        let resp = self.http
            .get("https://weibo.com/ajax/side/hotSearch")
            .header("User-Agent", "Mozilla/5.0 ...")
            .send()
            .await?
            .json::<serde_json::Value>()

        let mut hotspots = Vec::new();
        if let Some(data) = resp["data"]["realtime"].as_array() {

                    hot_score: item["num"].as_u64().unwrap_or(0),

## 4. 请求频率控制

每个数据源独立限速，避免触发反爬：

```rust
pub struct SourceRateLimiter {
    /// 每个数据源的最后请求时间
    last_request: Mutex<HashMap<HotspotSource, Instant>>,
    /// 最小请求间隔（默认 60 秒）
    min_interval: Duration,
}

impl SourceRateLimiter {
    pub fn new() -> Self {
        Self {
            last_request: Mutex::new(HashMap::new()),
            min_interval: Duration::from_secs(60),
        }
    }

    pub async fn acquire(&self, source: &HotspotSource) {
        let wait = {
            let map = self.last_request.lock().unwrap();
            let now = Instant::now();
            if let Some(last) = map.get(source) {
                let elapsed = now.duration_since(*last);
                if elapsed < self.min_interval {
                    Some(self.min_interval - elapsed)
                } else {
                    None
                }
            } else {
                None
            }
        };

        if let Some(wait) = wait {
            tokio::time::sleep(wait).await;
        }

        let mut map = self.last_request.lock().unwrap();
        map.insert(source.clone(), Instant::now());
    }
}
```

## 5. 缓存策略

| 参数 | 值 | 说明 |
|------|-----|------|
| 缓存有效期 | 5 分钟 | 热点变化频率适中 |
| 降级策略 | 返回过期缓存 | 单源失败不阻断 |
| 强制刷新 | 提供 `force_refresh` 参数 | 用户手动刷新时跳过缓存 |
| 最大缓存条数 | 500 条 | 避免内存占用过大 |
