# 用户模块详细设计

> 模块路径：`self-media-core/src/user` | 依赖：`self-media-crypto`、`self-media-db`

## 1. 功能清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 本地账号注册 | 用户名 + 密码，密码强度校验 | P0 |
| 登录验证 | 密码校验 → 签发 JWT → 创建会话 | P0 |
| 密码修改 | 旧密码验证 → 新密码 → 重加密所有敏感数据 | P0 |
| 会话管理 | JWT Token 本地签发，7 天有效期 | P0 |
| 会话续期 | 活跃用户自动续期 | P1 |
| 登出 | 清除会话记录 | P0 |

## 2. 输入校验规则

| 字段 | 规则 | 错误码 |
|------|------|--------|
| 用户名 | 长度 3-32，仅允许字母/数字/下划线 | AUTH_004 |
| 密码 | 长度 8-64，必须包含大小写字母 + 数字 | AUTH_005 |
| 密码确认 | 必须与密码一致 | AUTH_005 |

```rust
pub fn validate_username(username: &str) -> Result<(), AppError> {
    if username.len() < 3 || username.len() > 32 {
        return Err(AppError::validation(AUTH_004, "用户名长度需在 3-32 之间"));
    }
    if !username.chars().all(|c| c.is_alphanumeric() || c == '_') {
        return Err(AppError::validation(AUTH_004, "用户名仅允许字母、数字、下划线"));
    }
    Ok(())
}

pub fn validate_password(password: &str) -> Result<(), AppError> {
    if password.len() < 8 || password.len() > 64 {
        return Err(AppError::validation(AUTH_005, "密码长度需在 8-64 之间"));
    }
    let has_upper = password.chars().any(|c| c.is_uppercase());
    let has_lower = password.chars().any(|c| c.is_lowercase());
    let has_digit = password.chars().any(|c| c.is_numeric());
    if !(has_upper && has_lower && has_digit) {
        return Err(AppError::validation(AUTH_005, "密码必须包含大小写字母和数字"));
    }
    Ok(())
}
```

## 3. 数据模型

### 3.1 表结构

```sql
CREATE TABLE users (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    username      TEXT NOT NULL UNIQUE,
    password_hash TEXT NOT NULL,
    salt          TEXT NOT NULL,
    created_at    TEXT NOT NULL DEFAULT (datetime('now')),
    updated_at    TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE TABLE sessions (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id     INTEGER NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    token       TEXT NOT NULL UNIQUE,
    expires_at  TEXT NOT NULL,
    created_at  TEXT NOT NULL DEFAULT (datetime('now'))
);

-- 过期会话清理索引
CREATE INDEX idx_sessions_expires ON sessions(expires_at);
```

## 4. 核心逻辑

### 4.1 UserService

```rust
pub struct UserService {
    db: SqlitePool,
    system_key: SystemKey,  // 系统密钥，用于 JWT 签名
}

impl UserService {
    pub fn new(db: SqlitePool, system_key: SystemKey) -> Self {
        Self { db, system_key }
    }

    /// 注册
    pub async fn register(&self, username: &str, password: &str) -> Result<(User, Session), AppError> {
        validate_username(username)?;
        validate_password(password)?;

        // 检查用户名是否已存在
        let existing = sqlx::query_scalar!(
            "SELECT COUNT(*) FROM users WHERE username = ?",
            username
        )
        .fetch_one(&self.db)
        .await?;
        if existing > 0 {
            return Err(AppError::auth(AUTH_001, "用户名已存在"));
        }

        let salt = generate_salt();
        let password_hash = hash_password(password, &salt)?;
        let user = sqlx::query_as!(
            User,
            "INSERT INTO users (username, password_hash, salt) VALUES (?, ?, ?) RETURNING *",
            username, password_hash, salt
        )
        .fetch_one(&self.db)
        .await?;

        let session = self.create_session(user.id).await?;
        Ok((user, session))
    }

    /// 登录
    pub async fn login(&self, username: &str, password: &str) -> Result<Session, AppError> {
        let user = sqlx::query_as!(
            User,
            "SELECT * FROM users WHERE username = ?",
            username
        )
        .fetch_optional(&self.db)
        .await?
        .ok_or(AppError::auth(AUTH_002, "用户名或密码错误"))?;

        if !verify_password(password, &user.password_hash, &user.salt)? {
            return Err(AppError::auth(AUTH_002, "用户名或密码错误"));
        }

        self.create_session(user.id).await
    }

    /// 密码修改（含重加密）
    pub async fn change_password(
        &self,
        user_id: i64,
        old_password: &str,
        new_password: &str,
    ) -> Result<(), AppError> {
        validate_password(new_password)?;

        let user = sqlx::query_as!(User, "SELECT * FROM users WHERE id = ?", user_id)
            .fetch_one(&self.db)
            .await?;

        if !verify_password(old_password, &user.password_hash, &user.salt)? {
            return Err(AppError::auth(AUTH_002, "旧密码错误"));
        }

        // 1. 用旧密码派生旧用户密钥，解密所有敏感数据
        let old_user_key = UserKey::derive_from_password(old_password, &user.salt)?;

        // 2. 生成新 salt 和新密码哈希
        let new_salt = generate_salt();
        let new_password_hash = hash_password(new_password, &new_salt)?;

        // 3. 用新密码派生新用户密钥
        let new_user_key = UserKey::derive_from_password(new_password, &new_salt)?;

        // 4. 重加密所有 API Keys
        self.reencrypt_api_keys(user_id, &old_user_key, &new_user_key).await?;

        // 5. 重加密所有平台配置中的 Cookie
        self.reencrypt_platform_cookies(user_id, &old_user_key, &new_user_key).await?;

        // 6. 更新用户密码和 salt
        sqlx::query!(
            "UPDATE users SET password_hash = ?, salt = ?, updated_at = datetime('now') WHERE id = ?",
            new_password_hash, new_salt, user_id
        )
        .execute(&self.db)
        .await?;

        // 7. 清除所有现有会话，强制重新登录
        sqlx::query!("DELETE FROM sessions WHERE user_id = ?", user_id)
            .execute(&self.db)
            .await?;

        Ok(())
    }

    /// 验证会话
    pub async fn validate_session(&self, token: &str) -> Result<User, AppError> {
        let session = sqlx::query_as!(
            Session,
            "SELECT * FROM sessions WHERE token = ? AND expires_at > datetime('now')",
            token
        )
        .fetch_optional(&self.db)
        .await?
        .ok_or(AppError::auth(AUTH_003, "会话已过期"))?;

        let user = sqlx::query_as!(User, "SELECT * FROM users WHERE id = ?", session.user_id)
            .fetch_one(&self.db)
            .await?;

        // 活跃会话自动续期（剩余有效期 < 1 天时续期到 7 天）
        self.maybe_renew_session(&session).await?;

        Ok(user)
    }

    /// 登出
    pub async fn logout(&self, token: &str) -> Result<(), AppError> {
        sqlx::query!("DELETE FROM sessions WHERE token = ?", token)
            .execute(&self.db)
            .await?;
        Ok(())
    }

    // ---- 内部方法 ----

    async fn create_session(&self, user_id: i64) -> Result<Session, AppError> {
        let token = self.system_key.generate_jwt(user_id)?;
        let session = sqlx::query_as!(
            Session,
            "INSERT INTO sessions (user_id, token, expires_at) VALUES (?, ?, datetime('now', '+7 days')) RETURNING *",
            user_id, token
        )
        .fetch_one(&self.db)
        .await?;
        Ok(session)
    }

    async fn maybe_renew_session(&self, session: &Session) -> Result<(), AppError> {
        // 简化：每次验证时检查，剩余 < 1 天则续期
        sqlx::query!(
            "UPDATE sessions SET expires_at = datetime('now', '+7 days') \
             WHERE id = ? AND expires_at < datetime('now', '+1 day')",
            session.id
        )
        .execute(&self.db)
        .await?;
        Ok(())
    }

    async fn reencrypt_api_keys(
        &self,
        user_id: i64,
        old_key: &UserKey,
        new_key: &UserKey,
    ) -> Result<(), AppError> {
        let keys = sqlx::query!(
            "SELECT id, encrypted_key FROM api_keys WHERE user_id = ?",
            user_id
        )
        .fetch_all(&self.db)
        .await?;

        for key_row in keys {
            let plaintext = old_key.decrypt(&key_row.encrypted_key)?;
            let new_encrypted = new_key.encrypt(&plaintext)?;
            sqlx::query!(
                "UPDATE api_keys SET encrypted_key = ? WHERE id = ?",
                new_encrypted, key_row.id
            )
            .execute(&self.db)
            .await?;
        }
        Ok(())
    }

    async fn reencrypt_platform_cookies(
        &self,
        user_id: i64,
        old_key: &UserKey,
        new_key: &UserKey,
    ) -> Result<(), AppError> {
        let configs = sqlx::query!(
            "SELECT id, cookies FROM platform_configs WHERE user_id = ? AND cookies IS NOT NULL",
            user_id
        )
        .fetch_all(&self.db)
        .await?;

        for config in configs {
            if let Some(cookies) = &config.cookies {
                let plaintext = old_key.decrypt(cookies)?;
                let new_encrypted = new_key.encrypt(&plaintext)?;
                sqlx::query!(
                    "UPDATE platform_configs SET cookies = ? WHERE id = ?",
                    new_encrypted, config.id
                )
                .execute(&self.db)
                .await?;
            }
        }
        Ok(())
    }
}
```

### 4.2 登录流程

```
用户输入用户名+密码
        │
        ▼
  输入校验 ◄── 失败 → AUTH_004/AUTH_005
        │
        ▼
  查询用户 ◄── 不存在 → AUTH_002
        │
        ▼
  验证密码 ◄── 错误 → AUTH_002
        │
        ▼
  派生用户密钥 (UserKey)
        │
        ▼
  系统密钥签发 JWT (SystemKey)
        │
        ▼
  写入 sessions 表
        │
        ▼
  返回 Session + 用户密钥缓存到内存
```
