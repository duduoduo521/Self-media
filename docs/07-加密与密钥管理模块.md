# 加密与密钥管理模块详细设计

> 模块路径：`self-media-crypto` | 依赖：`aes-gcm`、`argon2`、`jsonwebtoken`、`subtle`、`base64`

---

## 1. 设计思路

### 1.1 双层密钥架构

原始方案使用单一主密钥（由用户密码派生），存在以下问题：

1. **JWT 验证死循环**：服务重启后，无用户密码则无法派生主密钥，已签发的 JWT 无法验证
2. **多用户密钥冲突**：Web 端需同时服务多个用户，不同用户应派生出不同密钥

**修复方案**：双层密钥架构

```
┌──────────────────────────────────────────────────┐
│                  系统密钥 (SystemKey)              │
│  ─ 来源：随机生成，持久化加密存储                    │
│  ─ 用途：JWT 签名/验证                            │
│  ─ 生命周期：应用级别，跨用户、跨会话                │
│  ─ 加密存储：使用机器指纹或环境变量加密              │
└──────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────┐
│                  用户密钥 (UserKey)                │
│  ─ 来源：用户密码 + Salt → Argon2id 派生           │
│  ─ 用途：加密/解密用户敏感数据（API Key、Cookie）    │
│  ─ 生命周期：用户会话级别，登录时派生                │
│  ─ 存储：仅存在于内存，不持久化                     │
│  ─ 缓存：Web 端通过 UserKeyCache 缓存             │
└──────────────────────────────────────────────────┘
```

## 3. 系统密钥 (SystemKey)

### 3.1 设计

```rust
use aes_gcm::{Aes256Gcm, Nonce};
use base64::{engine::general_purpose::STANDARD as BASE64, Engine};
use jsonwebtoken::{encode, decode, Header, Algorithm, EncodingKey, DecodingKey, Validation};
use rand::rngs::OsRng;
use rand::RngCore;

/// 系统密钥：用于 JWT 签名/验证
/// 随机生成，加密持久化，应用级别生命周期
pub struct SystemKey {
    key_bytes: [u8; 32],
    signing_key: EncodingKey,
    decoding_key: DecodingKey,
}

impl SystemKey {
    /// 生成新的系统密钥
    pub fn generate() -> Self {
        let mut key_bytes = [0u8; 32];
        OsRng.fill_bytes(&mut key_bytes);
        Self::from_bytes(&key_bytes)
    }

    /// 从原始字节构建
    fn from_bytes(key_bytes: &[u8; 32]) -> Self {
        Self {
            key_bytes: *key_bytes,
            signing_key: EncodingKey::from_secret(key_bytes),
            decoding_key: DecodingKey::from_secret(key_bytes),
        }
    }

    /// 从加密存储加载系统密钥
    pub fn load(encrypted: &str, machine_key: &[u8]) -> Result<Self, AppError> {
        let combined = BASE64.decode(encrypted)?;
        if combined.len() < 12 {
            return Err(AppError::crypto(CRYPTO_001, "系统密钥密文格式错误"));
        }
        let nonce = Nonce::from_slice(&combined[..12]);
        let ciphertext = &combined[12..];
        let cipher = Aes256Gcm::new_from_slice(machine_key)?;
        let decrypted = cipher.decrypt(nonce, ciphertext)
            .map_err(|_| AppError::crypto(CRYPTO_001, "系统密钥解密失败"))?;
        if decrypted.len() != 32 {
            return Err(AppError::crypto(CRYPTO_001, "系统密钥长度错误"));
        }
        let mut key_bytes = [0u8; 32];
        key_bytes.copy_from_slice(&decrypted);
        Ok(Self::from_bytes(&key_bytes))
    }

    /// 加密持久化系统密钥
    pub fn save(&self, machine_key: &[u8]) -> Result<String, AppError> {
        let nonce = Aes256Gcm::generate_nonce(&mut OsRng);
        let cipher = Aes256Gcm::new_from_slice(machine_key)?;
        let ciphertext = cipher.encrypt(&nonce, self.key_bytes.as_slice())?;
        let mut combined = nonce.to_vec();
        combined.extend_from_slice(&ciphertext);
        Ok(BASE64.encode(&combined))
    }

    /// 签发 JWT
    pub fn generate_jwt(&self, user_id: i64) -> Result<String, AppError> {
        let expiration = Utc::now() + Duration::days(7);
        let claims = serde_json::json!({
            "sub": user_id,
            "exp": expiration.timestamp(),
            "iat": Utc::now().timestamp(),
        });
        let token = encode(&Header::new(Algorithm::HS256), &claims, &self.signing_key)?;
        Ok(token)
    }

    /// 验证 JWT
    pub fn verify_jwt(&self, token: &str) -> Result<i64, AppError> {
        let data = decode::<serde_json::Value>(
            token,
            &self.decoding_key,
            &Validation::new(Algorithm::HS256),
        )?;
        let user_id = data.claims["sub"]
            .as_i64()
            .ok_or(AppError::auth(AUTH_003, "无效的 JWT"))?;
        Ok(user_id)
    }
}

impl Drop for SystemKey {
    fn drop(&mut self) {
        self.key_bytes.fill(0);
    }
}
```

### 3.2 机器密钥来源

系统密钥的加密密钥（`machine_key`）来源优先级：

1. 环境变量 `SELF_MEDIA_MACHINE_KEY`（推荐生产环境）
2. 操作系统密钥链（macOS Keychain / Windows DPAPI）
3. 机器特征指纹（MAC 地址 + 主机名的 SHA-256）

## 4. 用户密钥 (UserKey)

### 4.1 设计

```rust
use argon2::{Argon2, Algorithm, Version, Params};
use subtle::ConstantTimeEq;

/// 用户密钥：用于加密/解密用户敏感数据
/// 由用户密码 + Salt 派生，仅存在于内存
#[derive(Clone)]
pub struct UserKey {
    key: [u8; 32],
}

impl UserKey {
    /// 从用户密码和 Salt 派生密钥（使用当前 argon2 crate API）
    pub fn derive_from_password(password: &str, salt_b64: &str) -> Result<Self, AppError> {
        let salt = BASE64.decode(salt_b64)?;

        let params = Params::new(
            ARGON2_MEM_COST,
            ARGON2_TIME_COST,
            ARGON2_LANES,
            Some(ARGON2_HASH_LENGTH),
        )?;

        let argon2 = Argon2::new(Algorithm::Argon2id, Version::V0x13, params);

        let mut key = [0u8; 32];
        argon2.hash_password_into(password.as_bytes(), &salt, &mut key)
            .map_err(|e| AppError::crypto(CRYPTO_001, &format!("密钥派生失败: {}", e)))?;

        Ok(Self { key })
    }

    /// AES-256-GCM 加密
    pub fn encrypt(&self, plaintext: &str) -> Result<String, AppError> {
        let nonce = Aes256Gcm::generate_nonce(&mut OsRng);
        let cipher = Aes256Gcm::new_from_slice(&self.key)?;
        let ciphertext = cipher.encrypt(&nonce, plaintext.as_bytes())
            .map_err(|e| AppError::crypto(CRYPTO_001, &format!("加密失败: {}", e)))?;
        let mut combined = nonce.to_vec();
        combined.extend_from_slice(&ciphertext);
        Ok(BASE64.encode(&combined))
    }

    /// AES-256-GCM 解密
    pub fn decrypt(&self, encrypted: &str) -> Result<String, AppError> {
        let combined = BASE64.decode(encrypted)?;
        if combined.len() < 12 {
            return Err(AppError::crypto(CRYPTO_001, "密文格式错误"));
        }
        let nonce = Nonce::from_slice(&combined[..12]);
        let ciphertext = &combined[12..];
        let cipher = Aes256Gcm::new_from_slice(&self.key)?;
        let plaintext = cipher.decrypt(nonce, ciphertext)
            .map_err(|_| AppError::crypto(CRYPTO_001, "解密失败，密钥可能已变更"))?;
        Ok(String::from_utf8(plaintext)
            .map_err(|_| AppError::crypto(CRYPTO_001, "解密结果非有效 UTF-8"))?)
    }

    /// 零化密钥内存（Drop 时自动调用）
    pub fn zeroize(&mut self) {
        self.key.fill(0);
    }
}

impl Drop for UserKey {
    fn drop(&mut self) {
        self.zeroize();
    }
}
```

> **修复说明**：
> 1. 使用当前 `argon2` crate (0.5+) 的 API：`Argon2::new()` + `hash_password_into()`
> 2. 实现 `Drop` trait，密钥离开作用域时自动零化内存
> 3. 独立为 `UserKey` 结构体，与 `SystemKey` 分离

## 5. 密码哈希与验证

```rust
/// 生成随机 Salt
pub fn generate_salt() -> String {
    let mut salt = [0u8; 16];
    OsRng.fill_bytes(&mut salt);
    BASE64.encode(&salt)
}

/// 密码哈希（使用当前 argon2 API）
pub fn hash_password(password: &str, salt_b64: &str) -> Result<String, AppError> {
    let salt = BASE64.decode(salt_b64)?;

    let params = Params::new(
        ARGON2_MEM_COST,
        ARGON2_TIME_COST,
        ARGON2_LANES,
        Some(ARGON2_HASH_LENGTH),
    )?;

    let argon2 = Argon2::new(Algorithm::Argon2id, Version::V0x13, params);

    let mut hash = [0u8; 32];
    argon2.hash_password_into(password.as_bytes(), &salt, &mut hash)?;

    Ok(BASE64.encode(&hash))
}

/// 密码验证（恒定时间比较，防时序攻击）
pub fn verify_password(password: &str, hash_b64: &str, salt_b64: &str) -> Result<bool, AppError> {
    let expected = BASE64.decode(hash_b64)?;
    let salt = BASE64.decode(salt_b64)?;

    let params = Params::new(
        ARGON2_MEM_COST,
        ARGON2_TIME_COST,
        ARGON2_LANES,
        Some(ARGON2_HASH_LENGTH),
    )?;

    let argon2 = Argon2::new(Algorithm::Argon2id, Version::V0x13, params);

    let mut actual = [0u8; 32];
    argon2.hash_password_into(password.as_bytes(), &salt, &mut actual)?;

    // 恒定时间比较，防止时序攻击
    Ok(expected.ct_eq(&actual).into())
}
```

> **修复说明**：
> 1. 使用 `subtle::ConstantTimeEq` 的 `ct_eq` 方法，而非 `==`
> 2. 使用当前 `argon2` crate API
> 3. `hash_password` 和 `verify_password` 使用统一的 `ARGON2_*` 常量
